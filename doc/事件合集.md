## pointermove

如果需要在移动端使用 pointermove 事件，可以通过下面两种方式之一，解决和移动端触摸事件冲突的问题：

- 在元素上添加 css 属性：

```css
touch-action: none;
```

- 在元素添加 touchStart 事件，并调用 `event.preventDefault()` 阻止浏览器默认行为

```js
const onTapStart = (e) => {
  e.preventDefault();
};
canvasRef.current.addEventListener("touchstart", onTapStart);
```
